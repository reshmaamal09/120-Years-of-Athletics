# 120-Years-of-Athletics

## Project Overview

This project analyzes 120 years of Olympic athletics data (1896–2016) to uncover trends, patterns, and insights related to athlete participation, country performance, and medal distribution.

The project focuses on transforming raw historical data into meaningful insights using data cleaning, preprocessing, and visualization techniques.

## Objectives
Analyze historical Olympic data across multiple years
Identify trends in athlete participation and performance
Understand medal distribution among countries
Create interactive dashboards for better decision-making

## Tools & Technologies Used
Microsoft Excel
Data cleaning
Handling missing values
Removing duplicates
Data preprocessing
Power BI
Data visualization
Dashboard creation
Interactive reports

## Data Processing Steps
Imported raw dataset into Excel
Cleaned data by:
Handling missing values
Removing duplicates
Standardizing data formats
Performed basic exploratory analysis
Loaded cleaned data into Power BI
Built interactive dashboards and visualizations

## Key Features of Dashboard
Medal count analysis by country
Year-wise performance trends
Athlete participation analysis
Sport-wise distribution
Top-performing countries visualization

## Key Insights
Certain countries consistently dominate Olympic medal counts
Athlete participation has significantly increased over time
Some sports show higher competition and medal concentration
Historical trends reveal shifts in global athletic performance

## Conclusion

This project demonstrates how historical data can be transformed into actionable insights using Excel and Power BI, helping in understanding long-term trends in global sports performance.
